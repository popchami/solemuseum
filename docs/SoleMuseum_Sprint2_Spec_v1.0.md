# SoleMuseum Sprint2 Spec v1.0

## Purpose

Sprint2 adds local sneaker data storage and basic CRUD.

## Scope

Implement SQLite storage, Shoe and Brand models, repositories, Riverpod providers, shoe registration, shoe detail, edit, delete, favorite toggle, and real data display on Home and Collection.

## Out of scope

Photos, image picker, wear logs, Pro purchase, backup, restore, sharing, cloud sync, login, Firebase, statistics, and Luxury theme.

## Completion criteria

The app can create, list, view, edit, delete, and favorite shoes using local SQLite data. Home shows the real collection count. Collection shows real shoes or an empty state. Sprint1 UI and Material 3 remain intact.
