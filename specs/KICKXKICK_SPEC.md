# Kick×Kick 仕様書 v1.1

## コンセプト

貼って、飾って、コレクション。

Kick×Kickはスニーカーを登録し、ステッカー化し、棚やボードに飾って楽しむアプリである。

単なる靴管理アプリではない。

- Collect（集める）
- Create（ステッカー化する）
- Exhibit（飾る）

を体験の中心に置く。

## 開発思想

Kick×Kickは売買アプリでも資産管理アプリでもない。

価格や購入先は記録できるが主役ではない。

最優先する判断基準は

『コレクターが楽しく飾れるか』

である。

## 最終定義

Collection = 博物館

Sticker = スクラップブック

靴が主役。

箱は補助演出。

PNGは共有機能ではなく素材化機能。

Kick×Kickは管理アプリではなく、スニーカーをデジタルステッカー化して飾るアプリである。
