# Kick×Kick Icon Specification v1.0

## 1. 目的

この仕様書は Kick×Kick のアプリアイコン制作条件を定義する。

対象:

- Google Play アイコン
- アプリアイコン
- スプラッシュ画面用アイコン
- Canva / Figma 制作用指示
- 画像生成AI用指示

## 2. 基本コンセプト

Kick×Kickのアイコンは、

> スニーカーをステッカー化して飾るアプリ

であることを一目で伝える。

モチーフ:

- 靴箱
- オリジナルスニーカー
- ステッカー感
- K×K ロゴ
- オレンジアクセント

## 3. アイコン構成

推奨構図:

```text
背景
↓
靴箱
↓
スニーカー
↓
白フチ
↓
太いオレンジ外フチ
↓
K×K ロゴ
```

## 4. 必須要素

- 靴箱
- オリジナルスニーカー
- K×K の文字
- 白フチ
- オレンジ外フチ
- モノクロ＋オレンジ配色

## 5. 避けること

- Nike / Adidas / New Balance などに見えるデザイン
- 実在ブランドのロゴ
- 実在スニーカーの特徴的すぎる形状
- キャラクターIPに見える要素
- 複雑すぎる背景
- 文字が小さすぎて読めない構成

## 6. カラー

基本:

```text
Black  : #111111
White  : #FFFFFF
Gray   : #F3F3F3
Orange : #FF7A1A
```

比率:

- 90% モノクロ
- 10% オレンジ

オレンジはアクセントとして使用する。
背景全面オレンジは避ける。

## 7. スニーカーのデザイン方針

スニーカーは特定ブランドに見えない完全オリジナルデザインにする。

方向性:

- 横向き
- シンプル
- ステッカー化しやすい形
- かっこいいがブランド不明
- 厚底すぎない
- ロゴなし

## 8. 靴箱のデザイン方針

靴箱は展示・コレクション感を出す補助要素。

方向性:

- シンプルな箱
- 正面または斜め前
- 箱に K×K ロゴ
- 箱が主役になりすぎない

## 9. ステッカー表現

Kick×Kickらしさとして、アイコンにもステッカー感を入れる。

表現:

- 白フチ
- 太いオレンジ外フチ
- 少し浮いたような見え方
- 角丸または柔らかいアウトライン

## 10. サイズ要件

制作元は大きめで作る。

推奨制作サイズ:

```text
1024 x 1024 px
```

必要に応じて以下へ縮小する。

```text
512 x 512 px
192 x 192 px
144 x 144 px
96 x 96 px
72 x 72 px
48 x 48 px
```

## 11. 背景

推奨:

- Off White
- Light Gray
- Black
- Dark Gray

避ける:

- 派手なグラデーション
- 写真背景
- ブランドを想起させる柄

## 12. アイコン内テキスト

使用可能:

```text
K×K
```

避ける:

```text
Kick×Kick
```

理由:

- 小さいサイズで読みにくい
- アイコン内文字量が多くなる

## 13. 画像生成AI用プロンプト方針

日本語指示例:


## 14. Canva / Figma 制作用メモ

- 1024px正方形で制作
- スニーカーと箱を中央配置
- 小さいサイズでも読めるようにK×Kは大きめ
- 細かすぎる装飾を避ける
- 最終確認は128px表示でも行う

## 15. 判定基準

良いアイコン:

- 一目でスニーカーアプリと分かる
- ステッカー感がある
- 特定ブランドに見えない
- 小さいサイズでも成立する
- オレンジが効いている

悪いアイコン:

- 靴管理アプリに見えるだけ
- 売買アプリに見える
- ブランドロゴに見える
- 情報量が多すぎる
- オレンジが強すぎる

## 16. 最重要ルール

Kick×Kickのアイコンは、管理ではなく「貼って、飾る」楽しさを表現する。

靴が主役。
箱は補助。
オレンジはアクセント。
実在ブランドには見せない。
