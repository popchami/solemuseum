# Collection 2.0 - Sticker Mode

## Concept

Change Collection from a management screen into a viewing screen.

Current:
- Card list
- Brand name
- Size
- Metadata first

Future:
- Sneaker first
- Visual collection wall
- Zoomable overview
- Sticker style presentation

---

## Display Modes

### Card Mode

Current layout.

Purpose:
- Management
- Search
- Detailed browsing

### Sticker Mode

New layout.

Purpose:
- Collection viewing
- Showcase
- Shelf feeling

---

## Sticker Specification

Input:
- Original photo
- Future cutout PNG

Sticker style:
- Transparent background
- White outline
- Soft shadow
- Unified scale

Goal:
Every sneaker looks like a collectible sticker.

---

## Collection Wall

Users should be able to zoom.

Levels:
- 2 columns
- 4 columns
- 6 columns
- 8 columns
- 10 columns

Zoom out should allow dozens or hundreds of sneakers to be viewed at once.

---

## Interaction

Tap sticker
→ Shoe Detail

Long press (future)
→ Quick actions
